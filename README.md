# Concrete Mix Calculator

This project is a simplified Concrete Mix Design Calculator developed as part of a research project. It estimates material quantities and predicts compressive strength using both engineering formulas and basic machine learning.

---

## Features

- Calculate concrete material quantities (cement, sand, aggregate, water)
- Estimate compressive strength using engineering formulas
- Machine Learning-based strength prediction (Linear Regression)
- Simple Graphical User Interface (GUI) using Tkinter
- Modular code structure for scalability

---


---

## Inputs

- Concrete volume (m³)
- Mix ratio (cement:sand:aggregate)
- Water-cement ratio
- Material quantities (for ML model)

---

## Outputs

- Cement (bags)
- Sand volume (m³)
- Aggregate volume (m³)
- Water requirement (liters)
- Predicted compressive strength (MPa)
